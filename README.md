# x-cakefest21
- This repository is the sample code for my presentation "Components Reconsidered" at [CakeFest2021](https://cakefest.org/).

## My Presentation
- T.B.D

## Setup

```bash
$ git clone git@github.com:itosho/x-cakefest21.git
$ cd x-cakefest21
$ docker-compose up -d
```

Access to http://localhost:8099
